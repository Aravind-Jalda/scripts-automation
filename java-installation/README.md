# Java 21 Installation Script (Ubuntu)

## 📌 Objective
This script automates:
- Removal of all existing Java versions
- Installation of OpenJDK 21
- Configuration of JAVA_HOME

---

## ⚙️ Prerequisites

- Ubuntu OS (18.04 / 20.04 / 22.04 or later)
- Sudo privileges
- Internet connectivity

---

## 📂 Files

- `install_java21.sh` → Main automation script

---

## 🚀 Usage

### Step 1: Make script executable
```bash
chmod +x install_java21.sh
